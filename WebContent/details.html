Smart Irrigation App
PRODUCTS IOT MOBILE DEVOPS

Market problem is water conservation with optimally irrigating the crop and enhancing the crop productivity. Over- or under-watering :- adversily affects crop yields and wastes water (think dry/drought areas). Farmers usually follow an irrigation schedule, whick is manually decided based on knowledge.

Smart irrigation system aims to address the problem with over / uns=der watering using IoT device to monitor crop profile attributes such as humidity, moisture content and use edge computing devices to capture sensor data and determine the optimal irrigation schedule. Historical Irrigation patterns for different croops can be used for the future, which is more manual & based on farmers knowledge or experiance in todays world

Applicatiion architecture uses Bluemix IoT foundation platform. MVC - Architecture drafted by Manoharan Ramasamy iin the architecture overview. Communication protocol over REST & MQTT with Edge computing layer and IoT sensor devices. Use of relational data store. Plantation & Irrigation schedule, Farms & Crops information. Sensors integrated with the application runtime. Arduino & Edge computing application platforms. Bluemix DevOps pipeline for biuls & deployment. GIT Hub for source code version management. Eclipse IDE for application build.
